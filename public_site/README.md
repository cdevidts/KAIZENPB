# Kaizen Web

Sitio estatico generado desde los markdown y el CSV del diagnostico Kaizen.

## Generar

```bash
node scripts/build_kaizen_web.mjs
```

## Abrir

Abrir `kaizen_web/index.html` en navegador.

## Contenido

- `kaizen_web/aprendizaje-kaizen.html`
- `kaizen_web/dashboard-kaizen.html`
- `kaizen_web/todos-los-problemas.html`
- `kaizen_web/merges.html`
- `kaizen_web/detalle/problemas/*.html` (23 exportados)
- `kaizen_web/detalle/merges/*.html` (7 exportados)
